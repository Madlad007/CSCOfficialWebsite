Canterbury Soccer Club Website

A modern, responsive website designed for Canterbury Soccer Club (NSW, Australia).
The project demonstrates a clean sports-focused design with a strong visual hero section, modern layout, and mobile-friendly structure.

The goal of this project was to build a professional sports club website template that is simple, fast, and visually engaging.

https://madlad007.github.io/CSCOfficialWebsite/

Project Overview

This website includes the essential sections needed for a sports club:

Modern hero section

Club about page

Team / club information

Contact section

Clean navigation menu

Mobile-friendly layout

The design focuses on:

simplicity

readability

strong visual impact

sports-style branding

Technologies Used

HTML5

CSS3

Responsive Layout Techniques

GitHub Pages (for hosting)

No frameworks were used in order to keep the project lightweight and easy to customize.

Features

• Modern sports-style homepage
• Responsive design for mobile and desktop
• Hero banner with strong visual presence
• Clean navigation layout
• Easy to customize for other sports clubs
• Hosted using GitHub Pages
